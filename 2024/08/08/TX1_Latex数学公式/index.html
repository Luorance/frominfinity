计算符号

  • 加号 ：+
  • 减号 ：-
  • 加减号 : \pm
  • 乘号 或 ：\times 或 *
  • 点 : \cdot
  • 除号 ： \div
  • 分数 ：\frac { 分子 } { 分母 }
  • 幂 ：x^2
  • 根号 ： \sqrt [ n ] { x }
  • : \propto

集合

  • 属于 ： \in
  • : \ni
  • 不属于： \notin
  • 空集： \emptyset
  • 并集： \cup
  • 交集： \cap
  • 子集： \subset
  • 非子集： \not\subset
  • : \nsubseteq
  • 真包含： \supset
  • 非包含： \not\supset
  • : \nsupseteq

逻辑符号

  • 与： \land 或 \wedge
  • 或： \lor 或 \vee
  • 否 : \neg
  • 蕴含： \rightarrow
  • 等价： \leftrightarrow
  • 存在量词： \exists
  • 全称量词： \forall
  • 非等价： \nleftrightarrow 或 \not\leftrightarrow
  • 非蕴含： \nrightarrow 或 \not\rightarrow
  • 差集：\setminus 或 \smallsetminus
  • 映射到：\mapsto
  • 恒等于：\equiv
  • 不等：\neq 或 \ne
  • 大于：>
  • 小于：<
  • 大于等于：\geq 或 \ge
  • 小于等于：\leq 或 \le

积分

  • 求和： \sum_{ i=1 } ^ { n } i
  • 极限：\lim_{x\to\infty} f(x)
  • 定积分： \int_{ a } ^ { b } f(x) dx
  • 不定积分：\int f(x) dx
  • 二重积分： \iint 或 \idotsint
  • 三重积分： \iiint 或 \iiidotsint
  • 导数： \frac {dy} {dx}
  • 二阶导数： y''
  • 偏导数 /： \partial 如 \frac { \partial^2 f } { \partial x^2 }
  • 微分： \frac { d } { dx }
  • 无穷大： \infty
  • 无穷小或： \delta 或 \vartriangle
  • 无穷级数： \sum_{ n=1 } ^ { \infty } \frac { 1 } { n^2 }
